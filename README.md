## 📋 What is skills-best-practices?

skills-best-practices is a simple tool designed to help you write professional skills for agents. It validates these skills using large language models (LLMs) and keeps the amount of information used lean and focused. Whether you want to set up new skills or keep your existing ones clear and efficient, this app guides you every step of the way.

You do not need to know coding to use this app. It runs on your Windows PC and gives you a clear path to create, check, and update agent skills.

## 🖥️ How to Use skills-best-practices

Once installed, follow these steps to start creating and managing skills:

### Open the Application

- Double-click the app icon on your desktop or find it in the Start menu under "skills-best-practices."

### Write a New Skill

1. Click **New Skill**.
2. Enter the name of the skill. For example, "Customer Support Response."
3. Type a short description explaining what the skill does.
4. Add example phrases agents might use.

### Validate Your Skill

1. After writing, click **Validate**.
2. The app will check the skill using a language model.
3. It will let you know if anything is unclear or needs improvement.

### Manage Skills

- View all your skills in the **Skill List**.
- Select any skill to edit, validate again, or delete.

## 🔧 Setup Tips and Best Practices  

- Write clear and concise skill descriptions to improve validation results.
- Use simple language when creating example phrases.
- Regularly validate your skills to keep them updated.
- Remove outdated or unused skills to keep your list lean.
- Restart the app after any update to apply changes.
